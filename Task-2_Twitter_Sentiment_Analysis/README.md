# Twitter Sentiment Analysis

This project was developed as part of my Machine Learning Internship at
Rhombix Technologies.

## Objective
To analyze Twitter data and classify tweets into Positive, Neutral, or Negative
sentiment using Natural Language Processing techniques.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF
- Logistic Regression

## Output
The model generates sentiment predictions and saves them in a CSV file.

## How to Run
1. Clone the repository
2. Install dependencies from requirements.txt
3. Run the notebook in Jupyter
